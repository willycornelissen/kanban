Bemvindo(a) ao MeuKanban.

Kanban é uma palavra de origem japonesa que significa literalmente registro ou placa visível. O MeuKanban é uma SPA (Single Page Application) que permite criar vários Kanbans para administrar e gerenciar qualquer empreendimento.

A aplicação usa o Local Storage para persistir os Kanbans criados e o framework AngularJS para organizar e estruturar o Javascript.

  • Permite especificar o número de colunas de um Kanban (2, 3, 4 ou 6)
  • Drag 'n' drop de cartões
  • Edição de cartões (cor e descrição)
  • Persistência local
